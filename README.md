# MAC Autoria — Minisite de Lançamento

Repositório com o minisite criado para divulgação do empreendimento **Autoria**, da **MAC Construtora**, localizado na Rua Fagundes Filho, no bairro da Saúde, em São Paulo.

---

## Sobre o empreendimento

**Autoria** é um lançamento residencial de alto padrão de lazer, estrategicamente posicionado a 500 m do metrô São Judas, com fácil acesso à Av. Paulista, Brooklin, Aeroporto de Congonhas, Expo Imigrantes e região do ABC.

- **Unidades:** de 45 m² a 73 m², em diversas configurações de planta
- **Valores:** a partir de R$ 599.000 (45 m²) e R$ 809.000 (73 m²)
- **Condomínio estimado:** R$ 350/mês
- **Financiamento:** aceito
- **Inclui unidades HMP** — condições especiais de financiamento para famílias com renda até 10 salários mínimos

## Como usar

O minisite é um arquivo HTML único, autocontido. Para visualizá-lo:

1. Faça o download do arquivo `index.html`
2. Abra-o diretamente no navegador, **ou**
3. Publique-o em qualquer serviço de hospedagem estática (Netlify, GitHub Pages, Vercel, etc.)

Não há dependências de servidor, banco de dados ou build.

---

## Publicação via GitHub Pages

Para publicar este minisite usando o próprio GitHub Pages:

1. Vá em **Settings > Pages** neste repositório
2. Em **Source**, selecione a branch `main` e a pasta `/root`
3. Clique em **Save**
4. O site estará disponível em `https://paulolozano6-dev.github.io/MAC-pages/`
